# Brand Guidelines

The visual design language for the Adsmith admin: color palette, typography, and spacing & layout rules.

## Overall theme

- The admin should be a light theme inspired by Google's Material Design 3.
- Hex values for our background layers: page bg `#F8F8F8`, secondary/panel bg `#E5EEF1`, elevated surface, and hover bg `#E5EEF1`.
- Text colors: primary `#888888`, secondary `#4E8AA9`, muted `#BBBBBB`.
- Border colors: default `#5BA3C7`, subtle `#50859F`.

## Brand / semantic colors

- Primary (brand) color `#3294D2` + its hover/active shades — used for buttons, links, active tab, selected row.
- Semantic colors: success `#8BC246`, danger/error `#CC3333`, warning `#E69A50`, info — and whether each needs a separate background vs. text shade for badges/alerts.

## Typography

- Confirm fonts: Lato (display) + Consolas, "Andale Mono", "Lucida Console", "Lucida Sans Typewriter", Monaco, "Courier New", "monospace" (mono).
- Type scale: sizes/weights/line-heights for page titles 26px, section headers 18px, body 14px, table-cell text 11px, and small/caption 11px.
- Base font size for dense data tables vs. body copy: 11px.

## Spacing & layout

- Base spacing unit (8px) and the allowed scale for margins/padding/gaps.
- Page gutters 20px + max content width for list 100%, detail 80%, and form pages 40%.
- Padding inside cards/panels 30px, table cells 8px (row height — compact vs. comfortable), and form fields 12px.
